# Create Page Workflow

**Trigger:** When a new public or admin page needs to be added to Next.js.

## Steps
1. Add a new folder in `app/(public)` or `app/(admin)`.
2. Create `page.tsx` (Server Component).
3. Add Metadata for SEO.
4. Fetch data on the server if possible.
5. Create UI components inside `components/features/` and import them.
