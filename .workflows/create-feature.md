# Create Feature Workflow

**Trigger:** When a new business feature is requested (e.g., "Add Blog functionality").

## Steps
1. **Database:** Design the Prisma schema and run migrations.
2. **Backend:** Create the NestJS module (`.workflows/create-module.md`).
3. **API:** Define the REST API endpoints (`.workflows/create-api.md`).
4. **Frontend API:** Set up TanStack Query hooks or Next.js fetch calls.
5. **UI:** Build the React components (`.workflows/create-page.md`).
6. **Review:** Ensure end-to-end functionality and follow coding standards.
