# Init Project Workflow

**Trigger:** When starting a new major project stack (Frontend/Backend).

## Steps
1. Use `npx create-next-app@latest` for the frontend.
2. Use `nest new` for the backend.
3. Remove boilerplate files and clean up standard directories.
4. Install necessary standard dependencies (Tailwind, Framer Motion, shadcn/ui, Prisma).
5. Ensure `tsconfig.json` follows strict settings.
6. Verify the applications build successfully.
