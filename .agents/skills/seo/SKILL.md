# SEO Skill

1. Use Next.js Metadata API for static and dynamic titles/descriptions.
2. Include Open Graph (OG) and Twitter Card tags.
3. Ensure semantic HTML structure (h1, h2, h3).
4. Generate a dynamic `sitemap.xml` and `robots.txt`.
5. Optimize all images with `next/image` and provide `alt` text.
