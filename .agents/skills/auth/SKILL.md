# Auth Skill

1. Use Supabase Auth for user identity and session management on the frontend.
2. Use `@nestjs/jwt` and `@nestjs/passport` on the backend.
3. Secure sensitive endpoints using custom AuthGuards.
4. Manage roles if necessary via JWT claims.
