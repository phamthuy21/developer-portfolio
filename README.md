# IT Developer Portfolio

A premium, modern, and high-performance developer portfolio built with Next.js 15, NestJS, and Supabase.

## Project Structure
- `/frontend`: Next.js 15 App Router application (React 19, Tailwind CSS v4, shadcn/ui).
- `/backend`: NestJS application (Prisma, Swagger).
- `/docs`: Project documentation and architecture guidelines.

## Quick Start
1. Run `pnpm install` in the root directory.
2. Setup your environment variables (copy `.env.example` to `.env`).
3. Run `pnpm run dev` to start the frontend and backend concurrently.

## Documentation
Please refer to `docs/PROJECT.md` as the Single Source of Truth for this project.

## Local Docker Development
You can run the full stack locally via Docker:
```bash
docker-compose up -d
```
